# ORPHION_WD_05
Build an app that fetches weather data based on user location or search query. Display humidity, wind speed, and temperature.

## API KEY

To make sure the web app works properly you should use your own API KEY in the script.js file. <br>
This web app is using the `openweathermap` API.

`js/script`

`apiKey="YOUR_API_KEY_HERE"`
